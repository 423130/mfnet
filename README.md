# MFNet Reproduction on the ISPRS Potsdam Dataset

## Overview

This project reproduces and evaluates the MFNet semantic segmentation framework on the ISPRS Potsdam remote sensing dataset. The objective was to understand the implementation of modern computer vision architectures and assess their performance on high-resolution aerial imagery.

The original MFNet architecture combines SAM-based feature extraction with transformer-based segmentation techniques. In this work, the model was configured, trained, and evaluated on a different dataset to study its effectiveness for land-cover classification tasks.
